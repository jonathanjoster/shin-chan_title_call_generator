# しんちゃんタイトルコールジェネレータ

デモ: https://jonathanjoster.github.io/shin-chan_title_call_generator/

クレヨン○んちゃんとは何も関係がありません。

# 特徴
- なぜか既視感を感じさせるデザイン
- タイトルがコールされる (コールさせすぎると黙ります)

## 使い方

1. タイトルにしたい文字列を入れます (20文字制限)。

2. Speak ボタンを押します。
